## 💡 Philosophy

Firework is built on three core principles:

1. **🚀 Performance First** - Zero-cost abstractions, optimized for speed
2. **🎯 Developer Experience** - Ergonomic APIs, minimal boilerplate
3. **🔌 Extensibility** - Powerful plugin system, community-driven
